# Lab 15: Cisco Secure Firewall Management Center REST API

This folder contains a guided FMC API lab and a small Python client. Begin with [Lab15.md](Lab15.md).

The workflow authenticates, discovers the domain, follows pagination, inventories managed devices and access control policies, reports access rules, and performs a guarded CRUD lifecycle on one `LAB15-` network object. It deliberately does not deploy policy changes to managed firewalls.
